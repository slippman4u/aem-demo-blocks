# AEM Demo Blocks

A shared library of AEM Edge Delivery Services blocks for demo use.

## How to use a block

1. Copy the block folder into your project's `/blocks/` directory
2. Follow the block's own README.md for setup instructions
3. Add the block table to your page in DA.live
4. Create the required data sheet in your DA.live site
5. Publish and test

## Available blocks

| Block | Description |
|---|---|
| xray | AEP-style Real Time Customer Profile x-ray panel |
| shopthelook | Shop the look for retail apparel sales |
